# 🧮 Calculator — Windows Forms (C++/CLR)

A simple and minimal desktop calculator built using **C++/CLR** with **Windows Forms** in Visual Studio. It supports all basic arithmetic operations and keyboard input for a smooth user experience.

---

## ✨ Features

- ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
- ⌨️ Keyboard input support
- Clean and minimal UI design
- Built with C++/CLR and Windows Forms

## 🚀 How to Run / Install

### Prerequisites

Make sure you have the following installed:

- [Visual Studio 2026 Community](https://visualstudio.microsoft.com/) (or later)
- Workloads required:
  - ✅ **Desktop development with C++**
  - ✅ **.NET desktop development**

### Steps to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. **Open the project**
   - Launch Visual Studio
   - Click `File` → `Open` → `Project/Solution`
   - Select the `.sln` file from the cloned folder

3. **Build the project**
   - Press `Ctrl + Shift + B` or go to `Build` → `Build Solution`

4. **Run the application**
   - Press `F5` or click the ▶️ **Start** button in Visual Studio
